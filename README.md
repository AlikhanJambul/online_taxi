# Online Taxi

Мобильное приложение для заказа такси с микросервисным бэкендом и веб-панелью администратора. Дипломный проект.

## О проекте

Online Taxi — это полноценная платформа для заказа поездок, состоящая из мобильного приложения для пассажиров и водителей, серверной части на базе микросервисной архитектуры и веб-панели для управления платформой.

## Функциональность

### Пассажир
- Регистрация и авторизация
- Выбор точки подбора и места назначения на интерактивной карте (Яндекс Карты)
- Поиск адресов с подсказками (только в пределах города)
- Выбор точки маршрута перетаскиванием пина на карте
- Предварительная оценка стоимости поездки
- Заказ такси и отслеживание статуса в реальном времени
- Профиль с фотографией

### Водитель
- Регистрация с заполнением профиля и загрузкой фото автомобиля
- Ожидание верификации администратором
- Получение заявок на поездку
- Push-уведомления о новых заказах (Firebase)

### Администратор
- Просмотр списка всех пользователей платформы
- Проверка заявок водителей: просмотр фото и данных автомобиля
- Одобрение или отклонение водителей

## Архитектура

Бэкенд построен на микросервисной архитектуре. Сервисы общаются между собой через gRPC и RabbitMQ.

```
online_taxi/
├── api/proto/              # Protobuf-схемы
├── backend/
│   └── services/
│       ├── auth-service        # Авторизация, JWT, профиль, аватары
│       ├── driver-service      # Профиль водителя, геолокация
│       ├── trip-service        # Поездки и статусы
│       ├── matching-service    # Поиск подходящего водителя
│       ├── notification-service # Push-уведомления
│       └── admin-service       # REST API для панели администратора
├── admin/                  # Веб-панель администратора
└── online_taxi_mobile/     # Flutter-приложение
```

## Стек технологий

| Часть проекта | Технологии |
|---|---|
| Мобильное приложение | Flutter, Dart, Riverpod, Yandex MapKit, gRPC |
| Бэкенд | Go, gRPC, Clean Architecture |
| Админ-панель | React, TypeScript, Vite, Ant Design |
| База данных | PostgreSQL + PostGIS |
| Кэш / геолокация | Redis GEO |
| Очередь сообщений | RabbitMQ |
| Хранилище файлов | MinIO (S3) |
| Push-уведомления | Firebase Cloud Messaging |
| Контейнеризация | Docker, Docker Compose |
